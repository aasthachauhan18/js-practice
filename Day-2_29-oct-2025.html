Variables are of three type

  1. var

    2. its a universal scope and allow redeclare and reassign

  2. let

    3. its a block scope use only when we have to declare Variables in {}

    it allow us to reassign but dont allow us to redeclare

  3. const

    4. const is also ablockscope

    const can't allow to redeclare or reassign

    (var a) it is called declaration

    (a = 10) is called assignment

    (let a = 20;) this is known as initialization
